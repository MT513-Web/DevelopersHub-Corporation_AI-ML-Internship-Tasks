# AI/ML Engineering Internship - DevelopersHub Corporation

This repository contains the tasks and projects completed during my AI/ML Engineering Internship at DevelopersHub Corporation. The goal of this internship is to build a strong foundation in data science, machine learning models, and prompt engineering.

##  Tasks Completed

### Task 1: Exploring and Visualizing the Iris Dataset
* **Objective:** Perform Exploratory Data Analysis (EDA) on the classic Iris dataset.
* **Tools:** Pandas, Matplotlib, Seaborn.
* **Key Achievements:** Created scatter plots, histograms, and box plots to identify feature distributions and outliers.


### Task 2: Short-Term Stock Price Prediction
* **Objective:** Predict the next day's closing price for Apple Inc. (AAPL) using historical data.
* **Tools:** yfinance, Scikit-learn (Linear Regression).
* **Key Achievements:** * Implemented data shifting logic (`.shift(-1)`) to ensure real future price prediction.
    * Achieved a high R-squared score of 0.99.
    * Visualized Actual vs Predicted trends using line charts.
